# Mini To‑Do List – HTML / CSS / JS

Petit projet de liste de tâches minimaliste réalisé en HTML, CSS et JavaScript vanilla.  
Idéal pour débuter ou pour un mini projet de portfolio.

## Aperçu

Fonctionnalités :
- Ajouter une tâche via un champ texte et un bouton.
- Cocher une tâche pour la marquer comme terminée (texte barré).
- Supprimer une tâche de la liste.
- Design simple, sombre, léger et responsive.

## Technologies

- **HTML5** : structure de la page.
- **CSS3** : mise en forme et responsive.
- **JavaScript** vanilla : logique de la liste (ajout, suppression, “done”).
