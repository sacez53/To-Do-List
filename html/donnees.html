Gestion des
données

Consultez, exportez ou supprimez vos données — vous gardez le contrôle total.

01 Vos données en un coup d'œil
  • E-mail Utilisé pour la connexion. Modifiable depuis les Paramètres.
  • Nom d'utilisateur Pseudonyme libre, visible dans l'interface. Modifiable à tout moment.
  • Tâches Titres, notes, statuts, priorités et dates. Stockés dans Firestore sous votre UID Firebase.
  • Mot de passe Jamais stocké en clair — géré exclusivement par Firebase Authentication.
  • Localisation Non collectée. Aucun tracking géographique.
Données hébergées sur Firebase (Google), région Europe
Métadonnées techniques conservées par Firebase selon leur politique (30–180 jours)
02 Exporter mes tâches

Téléchargez une copie complète de toutes vos tâches au format JSON lisible. Ce fichier contient les titres, notes, statuts, priorités et dates d'échéance. Il peut être réimporté ou archivé.

Exportez également vos tâches au format CSV (tableur) — compatible Excel, LibreOffice Calc et Google Sheets.

03 Modifier mes informations

Toutes les données personnelles modifiables sont accessibles depuis la page Paramètres : nom d'utilisateur, adresse e-mail et mot de passe. Les modifications sont appliquées immédiatement dans Firebase.

→ Accéder aux Paramètres
04 Suppression des données

Vous pouvez supprimer uniquement vos tâches (en conservant votre compte), ou supprimer intégralement votre compte et toutes les données associées. Ces actions sont irréversibles.

Supprimer toutes les tâches sans supprimer le compte. Votre accès reste actif mais la liste sera vide.

Supprimer le compte et toutes les données (tâches, profil, authentification). L'accès sera immédiatement révoqué.
